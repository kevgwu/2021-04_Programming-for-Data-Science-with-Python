In this assignment, I try to predict an individuals' income based on census data. The following tasks are done in this assignment:

1. Feature exploration
2. Data preparation
3. Normalizing of numerical features from 0 to 1
4. Hot-encoding of categorical features
5. Shuffle & splitting of data
6. Fitting multiple models, evaluating for performance, and choosing a model.
7. Optimizing chosen model via grid search.
8. Evaluating the final model with test data.
9. Assessing for features required (selected) for performance using feature importance.
